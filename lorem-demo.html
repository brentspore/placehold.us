AI Lorem Text Generator

Unique, AI-generated placeholder text for your designs • Just like our image placeholders

Basic Usage

Generate placeholder text with a simple URL pattern:

placehold.us/lorem/[words|chars]/[number]/[style]
lorem/words/50

50 words of casual text

lorem/chars/200

200 characters

lorem/words/100/professional

100 professional words

lorem/chars/500/technical

500 technical characters

Available Styles

Style Description Example URL
casual Conversational, everyday language (default) lorem/words/50
professional Business-appropriate, formal tone lorem/words/50/professional
technical Tech jargon, documentation style lorem/words/50/technical
creative Imaginative, storytelling language lorem/words/50/creative
formal Academic, sophisticated writing lorem/words/50/formal
playful Fun, lighthearted tone lorem/words/50/playful

Output Formats

Add a format parameter to get different output types:

Format Description Example URL
text Plain text (default) lorem/words/50/casual/text
json JSON object with text property lorem/words/50/casual/json
html Wrapped in <p> tags lorem/words/50/casual/html

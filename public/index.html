Welcome to placehold.us
A quick and simple image placeholder service.

How does it work? Just put your image size after our URL and you'll get a placeholder.
Like this: https://placehold.us/350x150
You can also use it in your code, like this:
<img src="https://placehold.us/350x150">
Have fun!

Image Formats

.svg, .png, .jpg, .jpeg, .gif

Adding an image file extension will render the image in the proper format. Image format is optional and the default is SVG (fast, scalable, and small file size). You can also request PNG, JPG, JPEG, or GIF formats. The image extension can go at the end of any option in the url

https://placehold.us/300.png/09f/fff
https://placehold.us/300/09f.jpg/fff
https://placehold.us/300/09f/fff.gif

Custom Text

&text=Hello+World

Custom text can be entered using a query string at the very end of the url. This is optional, default is the image dimensions (300Ã250). A-z (upper and lowercase), numbers, and most symbols will work just fine.

Note: Spaces become +
https://placehold.us/300&text=placehold.us+rocks!

Colors

background color / text color

Colors are represented as hex codes (#ffffff is white)

Colors always follow the dimensions
https://placehold.us/250/ffffff/000000
not
https://placehold.us/ffffff/250/000000

The first color is always the background color and the second color is the text color.

Size

width x height

Height is optional, if no height is specified the image will be a square.

Example: https://placehold.us/300
Note: Size must be the first option in the url
